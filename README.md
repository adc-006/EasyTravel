# Easy-Travel

本项目使用vue-cli2.0搭建，vue.js构建实现

## 项目介绍

### 1.项目启动

```
npm run dev
```

#### 2.项目简介

易旅旅游网站，实现了首页、详情页以及城市选择的页面开发，用户可以查看全国各地城市的景点和景点详情

前端使用vue实现

后端数据使用axios进行交互
